# pi-remote frontend

React + Vite + Tailwind v4 + shadcn-style Radix components.

**Full UI docs:** [`../docs/UI.md`](../docs/UI.md)

```bash
pnpm install   # from frontend/
pnpm dev       # HMR dev server
pnpm build     # outputs to ../public/
```

From repo root:

```bash
pnpm run dev:ui
pnpm run build:ui
```
